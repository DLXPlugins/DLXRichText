# DLXRichText
Demonstrates in two branches how to convert RichText to InnerBlocks

## Folder Structure

```ascii
.
└── dlxrichtext/
    ├── dlxrichtext.php
    ├── package.json
    ├── build
    └── src/
        ├── index.js
        └── blocks/
            └── dlxrichtext/
                ├── block.js
                ├── edit.js
                └── block.json
```
